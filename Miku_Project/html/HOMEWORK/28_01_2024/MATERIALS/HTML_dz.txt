створити 4 сторінки
1. Робота зі списками
		згідно прикладу створити 2 блока зі списками. для кожного блоку списку додати класс і кастомізувати CSS 
2. Робота з блоками
		згідно прикладу створити 3 div (1 текст, 2 список, 3 додати зображення). для кожного блоку додати класс і кастомізувати CSS 
3. Робота з кнопками
		згідно прикладу створити 3 кнопки. кастомізувати, колір, розмір літер, закруглення
4. Сайт
	згадно прикладу створити сайт, повинен складатися за хедера, футера, блоків з текстом та малюнком
